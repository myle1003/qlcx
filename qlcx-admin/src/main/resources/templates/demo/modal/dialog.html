Set options

The relevant functions of the modal window can be realized through data binding or Javascript. If you use data binding, you can add data- to the element, such as data-backdrop=""< /code>.

Name Type default value Description
backdrop boolean or string 'static' true Mask layer, or use 'static' to specify that the mask layer is not associated with closing the modal window
keyboard boolean true Exit the modal window when pressing Esc
show boolean true Display modal window after initialization is complete
remote path false

Recommended to use data binding, or use jQuery.load

Remote URL example:

<a data-toggle="modal" href="remote.html" data-target="#modal">Click me</a>
